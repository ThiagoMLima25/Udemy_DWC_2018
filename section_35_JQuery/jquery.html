O que é JQUERY?

É uma biblioteca Java Script criado por Josh Resig

-> Acessar o DOM -> Adicionar efeitos visuais e animações -> Alterar a apresentação de uma página -> Buscar informações no servidor sem recarregar a página

Arquivo local ou via CDN(Link externo).

Principais Benefícios

-> Funciona em todos navegadores modernos -> Simplifica tarefas para criar sites para todas as mídias -> Focar no design -> Extensivos - Permite usar e criar plugins direcionados para fins especificos
